## 🧠 Sobre mim

Sou desenvolvedor fullstack com trajetória iniciada nos primórdios da web dinâmica, quando ASP3 e VBScript eram o estado da arte. Ao longo de mais de duas décadas, acompanhei e me adaptei a cada evolução do ecossistema web — sempre com foco em **entregar valor real para o negócio**.

Minha principal habilidade não é apenas escrever código, mas **entender o problema antes de propor a solução**. Tenho capacidade de transitar entre camadas de negócio, arquitetura e implementação, traduzindo necessidades complexas em sistemas funcionais, escaláveis e de fácil manutenção.

Além do desenvolvimento web, já explorei o mundo do hardware com **Arduino** e programação em **Linguagem C** — o que amplia minha visão sobre sistemas embarcados e computação de baixo nível.

## 💼 O que me diferencia

- **Visão de negócio**: entendo que software existe para resolver problemas reais — não apenas para funcionar tecnicamente.
- **Experiência transversal**: do front-end responsivo ao back-end robusto, passando por cloud, banco de dados e integração de sistemas.
- **Resolução de problemas**: capacidade consolidada de diagnosticar gargalos, propor refatorações estratégicas e tomar decisões técnicas sob pressão.
- **Histórico evolutivo**: vivenciei as diferentes eras do desenvolvimento web — de páginas estáticas a SPAs, microserviços e arquiteturas serverless.
- **Adaptabilidade**: cada projeto é único; a tecnologia é escolhida para servir a solução, não o contrário.
